# Log Analyzer in Python

## 📌 Overview

This project scans a log file and extracts all the 'Error' and 'Warning' messages, skipping empty or 'None' lines.
It save the result in a new file and provides a summary count.

## ⚙️ How to Run
1. Place your log file in the project folder (e.g., `sample_logs.txt`).
2. Run:
   ```bash
   python main.py
